MariaDB

O MariaDB Server é um sistema de gerenciamento de banco de dados relacional de código aberto de propósito geral. É um dos servidores de banco de dados mais populares do mundo, com usuários notáveis, incluindo Wikipedia , WordPress.com e Google . O MariaDB Server é lançado sob a licença de código aberto GPLv2 e tem garantia de permanecer de código aberto.

Numeração de versão:

Até o MariaDB 5.5, o MariaDB Server seguia o esquema de numeração de versão do MySQL, visando compatibilidade imediata com a mesma versão principal do MySQL.

Em 2012, para refletir a presença de um número crescente de recursos que não estavam disponíveis no MySQL, a numeração de versão do MariaDB Server divergiu , e o MariaDB lançou o 10.0, enquanto o MySQL lançou o 5.6. A versão de suporte de longo prazo atual é o MariaDB 10.6, enquanto a versão de suporte de curto prazo estável mais recente é o MariaDB 10.9.
