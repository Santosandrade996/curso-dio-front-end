Código de computador

O elemento de código

Exemplo de código de programação:

x = 5; y = 6; z = x + y;

O elemento de código não preserva espaços em branco e quebras de linha.

Para corrigir isso, você pode colocar o elemento code dentro de um elemento pre:

        
            x= 5;
            y =6;
            z= x + y;
