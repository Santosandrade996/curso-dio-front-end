O atibuto Start

por padrão, uma lista ordenada começará a contar a partir de 1. Use o atributo start para começar a contar a partir de um número especificado:

  1. Feijão
  2. Arroz
  3. Ovo
  1. Macarrão
  2. Carne Moida
  3. Molho Branco

Uma lista aninhada

As listas podem ser aninhadas(listas dentro de listas):

  1. Fejão
  2. Arroz
    1. Ovo
    2. Galinha assada
  3. Molho Vermelho

Uma lista de descrição

A dl tag define a lista de descrições, a tag define o termo (nome) e a tag descreve cada termo: dt dd dd

Café
-bebida quente preta
Leite
-bebida gelada branca
  • Use o elemento HTML dl para definir uma lista de descrição
  • Use o elemento HTML dt para definir o termo de descrição
  • Use o elemento HTML dd para descrever o termo em uma lista de descrição
