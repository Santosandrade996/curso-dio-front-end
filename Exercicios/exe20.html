Borda Css

A propiedade Css border define uma borda ao redor do elemento html

Preenchimento Css

A padding propiedade CSS define um preenchimento (espaço) entre o texto e a borda

Margem CSS

A margin propiedade CSS define uma margem(espaço) fora da borda

Link para CSS externo

Folhas de estilo externas podem ser referenciadas com a URL completo ou com um caminho relativo á pagina web atual.

Outro exemplo vincula a uma folha de estilo localizada na pasta html no site atual exemplo: link rel ="stylesheet" href="/html/styles.css"

Ultimo exemplo vincula a uma folha de estilo localizada na mesma pasta da página atual... Exemplo: link rel ="stylesheet" href ="styles.css"
