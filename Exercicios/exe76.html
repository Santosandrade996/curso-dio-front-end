Os elementos P e DIV são elementos de bloco e sempre começarão em uma nova linha e ocuparão toda a largura disponível (estendendo-se para a esquerda e para a direita o máximo possível).

Elementos de nivel de bloco html

main nav noscript ol p pre
section table tfoot ul video

O elemento SPAN é um elemento embutido e não começa em uma nova linha, ocupando apenas a largura necessária.

Aqui estão os elementos embutidos em HTML:

a abbr acronym b bdo big
br button cite code dfn em
i img input kbd label map
object output q samp script select
small span strong sub sup textarea
time tt var

Londres

Londres é a capital da Inglaterra. É a cidade mais populosa do Reino Unido, com uma área metropolitana de mais de 13 milhões de habitantes.

Situada às margens do Rio Tâmisa, Londres tem sido um importante assentamento há dois milênios, com sua história remontando à sua fundação pelos romanos, que a chamaram de Londinium.
