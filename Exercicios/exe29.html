Imagem de Fundo parte 2

Você pode especificar imagens de fundo para qualquer elemento HTML visível. Neste exemplo, a imagem de fundo é especificada para um elemento div. Por padrão, a background-image se repetirá na(s) direção(ões) onde for menor que o elemento onde for especificada. (Tente redimensionar a janela do navegador para ver como a imagem de fundo se comporta.)
