O elemento base

Observe que especificamos apenas um endereço relativo para a imagem. Como especificamos uma URL base na seção head, o navegador procurará a imagem em "https://www.w3schools.com/images/stickman.gif".

tag basica HTMLObserve que o link abre em uma nova janela, mesmo que não tenha o atributo target="_blank". Isso ocorre porque o atributo target do elemento base está definido como "_blank".
