Elemento HTML header

O header elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

Você pode ter vários header elementos em um documento HTML. No entanto, header não pode ser colocado dentro de um footer, address ou outro header elemento.

Elemento HTML footer

Footer elemento normalmente contém:

  • informações de autoria
  • informações de direitos autorais
  • informações de contato
  • mapa do site
  • voltar aos links principais
  • documentos relacionados

Elemento aside

O aside elemento define algum conteúdo além daquele em que é colocado (como uma barra lateral).

O aside conteúdo deve estar indiretamente relacionado ao conteúdo circundante.

Exemplo...

Minha família e eu visitamos o centro do Epcot neste verão. O clima estava bom, e o Epcot foi incrível! Tive um ótimo verão com minha família!

Centro Epcot
