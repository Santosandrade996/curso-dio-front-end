Imagem responsiva

Quando a propriedade CSS width é definida em um valor percentual, a imagem será dimensionada para cima e para baixo ao redimensionar a janela do navegador. Redimensione a janela do navegador para ver o efeito.

Imagem responsiva

"max-width:100%" impede que a imagem fique maior que seu tamanho original. No entanto, se você diminuir a janela do navegador, a imagem ainda será reduzida.

Redimensione a janela do navegador para ver o efeito.
