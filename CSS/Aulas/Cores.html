Cores

Existem várias formas de definirmos cores utiliando CSS:

  • Cores pé-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com hex-transparencia
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl(hue, saturation, lightness)
  • Hue (matiz): é o grau na roda de cores (de 0 e 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation (saturação):Valor percentual
    • 0% um tom cinza
    • 100% cor total
  • Lightness(Luminosidade): luminosidade da cor
    • 0% é preto
    • 100% é branco
